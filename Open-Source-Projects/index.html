Open Source Projects

Chromium

Projeto de navegador de código aberto que serve de base para o Google Chrome e outros navegadores modernos.

React

Biblioteca JavaScript para construir interfaces de usuário reativas, criada pelo Facebook.

Ubunto

Distribuição Linux baseada em Debian, voltada para usabilidade, segurança e atualização frequente.

Git

Sistema de controle de versões distribuído, eficiente para rastrear mudanças em projetos de software.

Android

Sistema operacional móvel desenvolvido pelo Google, usado em bilhões de dispositivos ao redor do mundo.

TypeScript

Superset do JavaScript que adiciona tipagem estática e recursos modernos para desenvolvimento robusto.

Tensorflow

Biblioteca de código aberto do Google para computação numérica e aprendizado de máquina com redes neurais.

Docusaurus

Gerador de sites estáticos focado em documentação técnica, criado pelo Facebook. ele é ideal para projetos.

Código aberto (do inglês Open Source) é o código-fonte que é disponibilizado gratuitamente para consulta, examinação, modificação e redistribuição. Os produtos incluem permissão para usar o código-fonte, documentos de design ou conteúdo do produto.

Fonte: https://pt.wikipedia.org/wiki/Código_aberto

Software livre é o software que concede liberdade ao usuário para executar, acessar e modificar o código fonte, e redistribuir cópias com ou sem modificações. Sua definição é estabelecida pela Free Software Foundation em conjunto com o projeto GNU.

Fonte: https://pt.wikipedia.org/wiki/Software_livre
